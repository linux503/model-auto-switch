### 3. 🏷️ **命名规范统一**

#### Python 脚本
- 核心功能: `module_name.py` (如 `model_ai_optimizer.py`)
- 工具脚本: 放在 `tools/` 目录
- 测试脚本: `test_*.py` 放在 `tests/` 目录

#### Shell 脚本
- 功能脚本: `function_name.sh`
- 工具脚本: 放在 `tools/` 目录
- 统一使用 `.sh` 扩展名

#### 文档文件
- 技术文档: 放在 `docs/` 目录
- 使用描述性文件名
- 统一 Markdown 格式

#### 配置文件
- 放在 `config/` 目录
- 使用 `.json` 或 `.config.js` 格式
- 提供默认配置和示例
